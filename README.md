
# SBIS и Аптеки: Обработка данных

## Описание
Скрипт предназначен для автоматизированной обработки CSV-файлов из системы **SBIS** и аптечных данных.  
Он выполняет следующие задачи:

- Чтение всех CSV-файлов из папки `Входящие` с поддержкой различных кодировок (`utf-8`, `cp1251`, `windows-1251`, `utf-8-sig`).  
- Объединение файлов SBIS в один DataFrame.  
- Фильтрация данных по типу документа (`СчФктр`, `УпдДоп`, `УпдСчфДоп`, `ЭДОНакл`) и удаление дубликатов по номеру документа.  
- Чтение CSV файлов аптек из папки `Аптеки/csv/correct/`.  
- Сравнение данных аптек с SBIS по номеру накладной и дате.  
- Формирование итоговых Excel-файлов с проверкой совпадения дат, сумм и номеров счетов-фактур.  
- Сохранение результатов в отдельные файлы для каждой аптеки в папке `Результат/YYYY-MM-DD/`.

## Установка
1. Клонируем репозиторий:
```bash
git clone <URL_репозитория>
cd <имя_проекта>
````
2. Создаём виртуальное окружение
```bash
python -m venv .venv
```
3. Активируем виртуальное окружение
```bash
- Windows
.venv\Scripts\activate
- Linux/MacOS
source .venv/bin/activate
```
4. Устанавливаем зависимости
```bash
pip install -r requirements.txt
```
5. Помещаем CSV-файлы SBIS в папку Входящие и CSV-файлы аптек в Аптеки/csv/correct/
6. Запускаем скрипт
```bash
python sbis_nalog_processing.py
```
